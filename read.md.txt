# Sales & Target Analysis Dashboard

## Project Overview

This project aims to analyze sales performance and compare actual sales against predefined targets using Power BI.

The sales dataset contains multiple business dimensions including Categories, Brands, Sub Channels, Customers, and Regions. A separate Target dataset was provided to track planned sales performance.

To enable accurate comparison between actual sales and target values, a custom composite key was created by combining:

- Category
- Brand
- Sub Channel

This key was then used to establish a relationship between the Sales and Target tables, allowing accurate target achievement calculations and performance analysis.

## Data Preparation

- Cleaned and transformed data using Power Query.
- Created a custom composite key to connect Sales and Target tables.
- Built data model relationships.
- Created DAX measures for KPIs and calculations.

## Dashboard Features

### Sales Dashboard

- Total Sales Analysis
- Sales by Category
- Sales by Channel
- Monthly Sales Trend
- Top Customers Analysis
- Pareto Analysis
- Stock Distribution by Region

### Target Dashboard

- Target Achievement %
- Sales vs Target Comparison
- Category Performance Analysis
- Brand Performance Analysis
- Sub Channel Performance Analysis
- Best Performing Brand
- Worst Performing Brand

## Key KPIs

- Total Sales
- Total Target
- Achievement Percentage
- Customer Count
- Best Brand
- Worst Brand

## Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling

## Business Value

This dashboard helps stakeholders monitor sales performance, track target achievement, identify top-performing brands and channels, and make data-driven business decisions.
