# Sales & Target Analysis Dashboard

## Project Overview

This project analyzes sales performance against predefined targets using Power BI.

The sales dataset contains multiple dimensions including Brands, Categories, Sub Channels, Customers, and Regions.

A separate Target table was provided containing planned sales targets. Since the Sales and Target datasets did not share a direct key, a composite key was created by combining:

- Category
- Brand
- Sub Channel

This key was used to establish a relationship between both tables and calculate target achievement accurately.

## Data Preparation

- Data Cleaning using Power Query
- Data Modeling
- Relationship Building
- Custom Composite Key Creation
- DAX Measures Development

## Dashboard Features

### Sales Dashboard

- Total Sales Analysis
- Monthly Sales Trend
- Top Customers Analysis
- Sales by Category
- Sales by Channel
- Pareto Analysis
- Stock Distribution Analysis

### Target Dashboard

- Target Achievement %
- Sales vs Target Comparison
- Category Performance Analysis
- Brand Performance Analysis
- Sub Channel Performance Analysis
- Best Performing Brand
- Worst Performing Brand

## Key KPIs

- Total Sales
- Total Target
- Achievement %
- Customer Count
- Best Brand
- Worst Brand

## Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling

## Business Value

This dashboard helps stakeholders monitor sales performance, track target achievement, identify top-performing brands and channels, and make data-driven business decisions.
